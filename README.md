# Aether Summit 2026

A premium, interactive event website built using React, Vite, and Tailwind CSS.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the local development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Structure
- `index.html` - Google Fonts integration and main HTML scaffolding.
- `tailwind.config.js` - Customized color scheme, animations, and typography tokens.
- `src/index.css` - Custom glassmorphic utilities and scrolling mechanics.
- `src/components/` - Modular design components (Navbar, Hero, Countdown, Speakers, Schedule, Register, Footer).
